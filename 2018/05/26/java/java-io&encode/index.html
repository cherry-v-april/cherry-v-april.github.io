I/O、Encode

Reader类是java的I/O中读字符的父类，而InputStream类是读字节的父类，InputStreamReader类就是关联字节到字符的桥梁，负责在I/O过程中读取字节到字符转换，而对具体字节到字符的解码实现是委托StreamDecoder去做。
